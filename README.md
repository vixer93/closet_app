# アプリケーション概要
自身のクローゼットの中にある洋服を一括管理するアプリケーションです。
服の色からどのような服とあうのか提案し、服選びの時間を短縮できます。

# 機能一覧
●服画像投稿機能

●画像認識機能(GoogleCloudVision)

●おすすめ機能

●スクレーピング機能

●ユーザー登録、ログイン機能
